### AlunoDAO (Manipulação de Aluno):

1. **Inserir Aluno:**
   - Método para inserir um novo aluno na tabela `aluno`.

2. **Listar Alunos:**
   - Método para recuperar todos os alunos da tabela `aluno`.

3. **Atualizar Aluno:**
   - Método para atualizar os dados de um aluno específico na tabela `aluno`.

4. **Excluir Aluno:**
   - Método para excluir um aluno específico da tabela `aluno`.

5. **Buscar Aluno por ID:**
   - Método para buscar um aluno específico com base no ID.
